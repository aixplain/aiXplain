# اختبارات مفتاح API

يحتوي هذا الدليل على اختبارات لوظائف مفتاح API في حزمة aiXplain SDK.

## المتطلبات الأساسية

لتشغيل هذه الاختبارات، تحتاج إلى:

1. مفتاح API إداري يمتلك صلاحيات لـ:
   - إنشاء مفاتيح API جديدة
   - تحديث مفاتيح API الحالية
   - حذف مفاتيح API
   - عرض قائمة مفاتيح API
   - عرض استخدام مفتاح API

2. خانات مفاتيح API المتاحة:
   - تقوم الاختبارات بإنشاء مفاتيح API وحذفها أثناء التنفيذ
   - تأكد من توفر خانة واحدة على الأقل لإنشاء مفتاح API
   - ستفشل الاختبارات إذا وصلت إلى الحد الأقصى لعدد مفاتيح API المسموح بها
