وحدة أداة الوكيل في aiXplain SDK.

توفر هذه الوحدة فئات الأدوات والوظائف التي تمكّن الوكلاء من التفاعل مع
الخدمات والنماذج ومصادر البيانات المختلفة.

حقوق النشر 2024 لمؤلفي aiXplain SDK

بموجب هذه الرخصة على أساس "كما هو"،
دون أي ضمانات أو شروط من أي نوع، سواء كانت صريحة أو ضمنية.
راجع الرخصة للاطلاع على الأذونات والقيود المحددة
المنصوص عليها فيها.
