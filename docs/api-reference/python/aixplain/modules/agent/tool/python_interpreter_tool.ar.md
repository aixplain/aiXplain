أداة مُفسِّر Python لوكلاء aiXplain SDK.

توفر هذه الوحدة أداة تتيح للوكلاء تنفيذ شيفرة Python
باستخدام مُفسِّر في بيئة مُتحكَّم بها.

حقوق النشر 2024 لمؤلفي aiXplain SDK

مرخَّص بموجب رخصة Apache، الإصدار 2.0 ("الرخصة")؛
لا يجوز لك استخدام هذا الملف إلا وفقًا للرخصة.
يمكنك الحصول على نسخة من الرخصة على:

     http://www.apache.org/licenses/LICENSE-2.0

ما لم يكن مطلوبًا بموجب القانون المعمول به أو تم الاتفاق عليه كتابيًا، يُوزَّع البرنامج
المقدَّم بموجب الرخصة على أساس "كما هو"،
دون أي ضمانات أو شروط من أي نوع، سواء كانت صريحة أو ضمنية.
راجع الرخصة للاطلاع على الأذونات والقيود المحددة
المنصوص عليها في الرخصة.
